Proyectos

Proyecto Sistema de Gestion de Ventas de Tickets de Recitales

Proyecto Grupal

Este proyecto consistió en desarrollar un sistema en C++ para la administración de conciertos, con el objetivo de gestionar usuarios, artistas, recintos, eventos y ventas de entradas de manera organizada. Se implementaron funcionalidades de ABML (Alta, Baja, Modificación y Listado) para todas las entidades, así como consultas de disponibilidad de entradas y generación de informes de ventas y asistencia. La interfaz gráfica facilita la interacción del usuario y la operación del sistema. Este trabajo grupal permitió coordinar módulos entre diferentes integrantes, aplicar programación orientada a objetos, manejo de archivos y garantizar la consistencia de la información entre las distintas clases relacionadas.

Proyecto Sistema de Gestion de Ventas de Tickets de Cine

Proyecto Individual

El sistema tiene como objetivo gestionar de manera eficiente la venta de entradas para funciones de cine. Permite administrar usuarios, películas, funciones y entradas disponibles, incluyendo operaciones de compra, todo a través de una interfaz gráfica intuitiva. El proyecto incluye funcionalidades de ABML (Alta, Baja, Modificación y Listado) para las distintas entidades, consultas de funciones disponibles, compra de entradas y generación de informes sobre ventas y asistencia. Este proyecto fue desarrollado íntegramente por mí, lo que me permitió aplicar de manera práctica la programación orientada a objetos, el manejo de archivos y la lógica de control de inventario, fortaleciendo mi autonomía en el desarrollo de sistemas completos.
